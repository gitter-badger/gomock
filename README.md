# gomock

gomock is command line tool which makes simple API mock server

# Install

```
go get github.com/hlts2/gomock
```

### Usage

```
$ gomock --help
Usage:
  gomock run [flags]

Flags:
  -h, --help         help for run
      --s string     set config file (default "config.yml")
      --set string   set config file (default "config.yml")
```
